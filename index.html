Tower

Tower is an application I made for users to list and attend events. Events have tickets that consumers can 'purchase', marking them as an attendee for the event. It is the first full stack application that I created on my own, and I used Node.js for the back end, and Vue.js for the front end. For the database, I MongoDb, an open source NoSQL database management program.

Keepr

Keepr is an application that I made for people to share, save, and group posts. The idea is that users can create vaults and posts. Posts are public to everyone, but vaults can be private. Vaults are a means of categorically storing posts. The back end of this project was built with C#, and the front end with Vue.js. This was my final project at Boise CodeWorks, and I passed, meeting all of the requirements.
